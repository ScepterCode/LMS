# Phase 4 Teacher Quick Start Guide

## For Form Teachers: Getting Started

Welcome to Phase 4! This guide helps you quickly start using your form teacher features.

---

## What's New in Phase 4?

As a form teacher, you now have:
- **Dashboard:** View all your assigned classes
- **Attendance:** Mark attendance daily for your class
- **Remarks:** Write end-of-term remarks for students
- **Reports:** Send term reports to parents
- **Grades:** View all grades for your class students

## Daily Tasks

### Task 1: Mark Attendance

**How to mark attendance:**

```
1. Go to your class
2. Click "Attendance" tab
3. Click "Mark Attendance" button
4. For each student:
   ✓ Check if Present
   ✗ Mark if Absent
   ⏰ Mark if Late (enter check-in time)
5. Click "Save"
```

**Attendance Status Options:**
- **Present** — Student was in class
- **Absent** — Student was not in class
- **Late** — Student arrived late (specify minutes)
- **Excused** — Justified absence

**Example:**
```
John Adeyemi      [✓ Present]
Mary Okonkwo      [✗ Absent] [Reason: Sick]
Grace Adekunle    [⏰ Late] [15 minutes]
```

---

### Task 2: Add Student Remarks (End of Term)

**How to add remarks:**

```
1. Go to your class
2. Click "Remarks" tab
3. Select a student
4. Enter your remark (max 500 characters)
5. Choose category:
   - Academic: Progress, performance, effort
   - Conduct: Behavior, discipline
   - Performance: Overall performance
   - General: General comments
6. Click "Save"
```

**Example Remark (Academic):**
> "John shows excellent progress in mathematics and actively participates in class discussions. He demonstrates strong problem-solving skills. Encouraged to maintain this performance."

**Character Count:** 
- Minimum: 10 characters
- Maximum: 500 characters
- Current: 156/500

## Reports to Parents

### How to Send Term Results

```
1. Go to your class
2. Click "Reports" tab
3. Click "Send Term Results"
4. Choose recipients:
   - "Send to all parents" (automatic)
   - "Select specific parents" (manual)
5. Click "Send"
```

**What gets sent:**
- Student's grades for all subjects
- Your remarks on student
- Attendance summary
- Overall performance

**Delivery Status:**
- **Pending** — Ready to send
- **Sent** — Delivered to parent
- **Read** — Parent opened report

## Important Rules & Reminders

### ⚠️ Attendance Rules

- ❌ **Cannot mark attendance for future dates**
  - Can only mark today or earlier
  - Cannot use future dates

- ✓ **Can mark attendance for:**
  - Today
  - Yesterday or earlier
  - Same term

- 🔄 **Can edit attendance:**
  - Click on previous attendance records
  - Correct student status
  - Save changes

### ⚠️ Remarks Rules

- ❌ **Cannot add remarks:**
  - For students not in your class
  - With empty text

- ✓ **Can add/edit remarks:**
  - For students in your class
  - Up to 500 characters
  - Can edit own remarks

- 📝 **Note:** System tracks who wrote each remark

### ⚠️ Report Sending Rules

- ❌ **Cannot send reports to:**
  - Parents not linked to class students
  - Outside current term

- ✓ **Can send:**
  - To parents of all class students
  - Multiple times per term
  - To specific parents if needed

## Tips & Best Practices

### Attendance Tips

✓ **Mark attendance immediately** after roll call (don't wait until end of day)

✓ **Be consistent** with check-in times for late students

✓ **Include reasons** for absences (medical, school trip, etc.)

✓ **Review previous** attendance before marking to spot patterns

### Remarks Tips

✓ **Be specific** - Mention strengths and areas for improvement

✓ **Be constructive** - Help student understand how to improve

✓ **Keep professional tone** - This goes to parents

✓ **Mention effort** - Not just grades, but also attitude and participation

Example:
> ❌ "John is good in math"
> ✓ "John demonstrates strong understanding of algebra and consistently attempts challenging problems. Could improve by asking more questions in class discussions."

### Report Tips

✓ **Send reports early** - Don't wait until term ends

✓ **Check delivery status** - Confirm parents received reports

✓ **Include remarks** - Reports are more meaningful with your comments

## Troubleshooting

### Issue: Can't mark attendance for a date

**Solution:** 
- Attendance date cannot be in the future
- Must be today or earlier
- Check your system date/time

### Issue: "Student not found" when adding remark

**Solution:**
- Student must be in your class
- Check if student transferred to another class
- Contact admin if unsure

### Issue: Report won't send

**Solution:**
- Verify parents are linked to students
- Check if term is current/active
- Contact your admin

### Issue: Can't see "Mark Attendance" button

**Solution:**
- You must be the form teacher
- Contact your admin if this is your class

## FAQ

**Q: Can I mark attendance late?**
A: Yes, but within the same term. You can edit previous attendance records.

**Q: Can another teacher see my remarks?**
A: No, only admins and you can see remarks you wrote.

**Q: What if a student misses many days?**
A: You can note the pattern in attendance, and contact parent or admin.

**Q: Can I change a student's grade?**
A: No, grades are entered by subject teachers. Contact them if there's an error.

**Q: Can I send custom reports?**
A: Currently, term results include grades and remarks. Contact admin for custom formats.

**Q: What happens if I send a report twice?**
A: Parents will receive it both times. This is allowed for resending.
